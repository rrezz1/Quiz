03
Kërkim teksti me LIKE
LIKE Strings
Ashtu si find() dhe startswith() në Python kërkojnë tekst brenda stringjeve, SQL ka operatorin LIKE për kërkim të modeleve.
E lehtë
Tabela: produktet
KolonaTipiShembull
produkt_idINT1
emriVARCHAR'Python Bootcamp'
kategoriaVARCHAR'Programim'

Python vs SQL: 'Python' in emri → SQL: emri LIKE '%Python%'
emri.startswith('Data') → SQL: emri LIKE 'Data%'
Çfarë duhet të bësh
Selekto emri, kategoria, cmimi nga produktet
Ku emri përmban fjalën 'Data' (kudo brenda emrit)
Sorto sipas cmimit nga i ulëti te i larti

04
Transformim i stringjeve
UPPER/LOWER CONCAT LEN
Funksionet e stringjeve të Python si upper(), lower(), len() kanë ekuivalentët e tyre direkte në SQL.
Mesatare
Ekuivalentet:
emri.upper() → UPPER(emri)
emri.lower() → LOWER(emri)
len(emri) → LEN(emri) ose LENGTH(emri)
'Hello' + ' ' + 'World' → CONCAT(col1, ' ', col2)
Çfarë duhet të bësh
Nga tabela students, krijo kolonë emri_madh = emri me shkronja të mëdha
Krijo kolonë profili = CONCAT(emri, ' nga ', qyteti)
Krijo kolonë gjatesia = gjatësia e emrit
Filtro vetëm ata ku gjatësia e emrit > 4 karaktere

05
Funksionet matematikore
ROUND ABS AVG
Ashtu si round(), abs() dhe math.ceil() në Python, SQL ka funksionet ROUND, ABS, CEILING, FLOOR për kontrollimin e numrave.
Mesatare
Tabela: shitjet
KolonaTipiShembull

Python → SQL:
round(x, 2) → ROUND(x, 2)
abs(x) → ABS(x)
sasia * cmimi → sasia * cmimi_unit
Çfarë duhet të bësh
Llogarit totali = sasia * cmimi_unit, i rrumbullakosur në 2 shifra decimale
Shfaq zbritja_abs = vlera absolute e zbritjes (pa shenjën minus)
Llogarit totali_neto = totali + zbritja (zbritja mund të jetë negative!)
Sorto nga totali_neto nga i larti te i ulëti

Tema 4 — Logjika & Kushtet

06
Deklaratat me kusht — CASE WHEN
CASE WHEN if-elif-else
Ekuivalenti i if-elif-else në Python është CASE WHEN ... THEN ... ELSE ... END në SQL. Kategorizo studentët sipas notës.
Mesatare
Python if-elif-else → SQL CASE WHEN:
if nota >= 9: 'Shkëlqyer'
elif nota >= 7: 'Mirë'
else: 'Ka nevojë për përmirësim'

Bëhet: CASE WHEN nota >= 9 THEN 'Shkëlqyer' WHEN nota >= 7 THEN 'Mirë' ELSE 'Ka nevojë' END
Çfarë duhet të bësh
Nga students, shfaq emri, nota
Krijo kolonë kategoria: nota ≥ 9 → 'Shkëlqyer', nota ≥ 7 → 'Mirë', nota ≥ 5 → 'Kalon', të tjera → 'Nuk kalon'
Sorto nga nota descending
Shkruaj SQL-in tënd
07
Operatorët logjik dhe IN
AND / OR IN NOT IN
Operatorët and, or, dhe operatori i anëtarësimit in / not in të Python korrespondojnë drejtpërsëdrejti me AND, OR, IN, NOT IN të SQL.
Mesatare
Python → SQL:
qyteti in ['Tiranë', 'Shkodër', 'Vlorë'] → qyteti IN ('Tiranë', 'Shkodër', 'Vlorë')
nota > 8 and mosha < 28 → nota > 8 AND mosha < 28
Çfarë duhet të bësh
Selekto emri, qyteti, nota, mosha nga students
Ku qyteti është IN ('Tiranë', 'Durrës', 'Vlorë')
AND nota >= 7.0
AND mosha NOT IN (17, 18) — (jo minorë që kaluan direkt)

08
Grumbullimi — GROUP BY
GROUP BY COUNT AVG
Ashtu si Dictionary në Python grupifikon të dhënat sipas çelësit, GROUP BY në SQL grumbullon rreshtat sipas vlerave të një kolone.
Mesatare
Ideja: Si do ta bëje në Python me dict: {'Tiranë': [9.5, 8.2, 7.1], 'Durrës': [8.8]}
Pastaj len(), sum()/len() për statistikat — SQL bën të njëjtën gjë me GROUP BY + COUNT() + AVG().
Çfarë duhet të bësh
Nga students, grupo sipas qyteti
Shfaq: qyteti, nr_studenteve = COUNT(*), nota_mesatare = AVG(nota) me 2 decimale
Filtro vetëm qytetet me nr_studenteve >= 2 (HAVING)
Sorto nga nota_mesatare DESC

10
Subquery — Funksion brenda funksioni
Subquery CTE Window Functions
Ashtu si funksionet e ndërlidhura në Python (output i një funksioni si input i tjetrit), subqueries dhe CTEs ju lejojnë të ndërtoni kueriet komplekse hap pas hapi.
E vështirë
Koncepti: Ashtu si filter(lambda x: x > avg(lista), lista) në Python, ne mund të krahasojmë çdo rresht me mesataren e grupit të tij. Kjo quhet Window Function: AVG(nota) OVER (PARTITION BY qyteti)
Çfarë duhet të bësh
Përdor CTE (WITH) për të llogaritur mesataren e notave për çdo qytet
Bashko CTE me students dhe shfaq çdo student me mesataren e qytetit të tij
Shto kolonën krahasim: 'Mbi mesatare' ose 'Nën mesatare' duke krahasuar notën individuale me mesataren e qytetit
Sorto sipas qyteti, pastaj nota DESC
