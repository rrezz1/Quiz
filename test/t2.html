GROUP BY – Shitjet sipas produktit

Gropo shitjet sipas emrit të produktit dhe gjej totalin për secilin produkt.

SELECT produkti, SUM(shuma) AS total, COUNT(*) AS nr_shitjeve FROM shitjet GROUP BY produkti HAVING SUM(shuma) > 1000 ORDER BY total DESC;

Missing Numbers – Gjej numrat që mungojnë

Nga tabela porositë me kolona ID, gjej të gjithë numrat ID që mungojnë në sekuencë.

-- Metoda me sequence generator (PostgreSQL) SELECT generate_series AS id_mungues FROM generate_series( (SELECT MIN(id) FROM porosite), (SELECT MAX(id) FROM porosite) ) WHERE generate_series NOT IN (SELECT id FROM porosite); -- Metoda me LEFT JOIN (punon kudo): SELECT t1.id + 1 AS id_mungues FROM porosite t1 LEFT JOIN porosite t2 ON t1.id + 1 = t2.id WHERE t2.id IS NULL ORDER BY id_mungues;

Formula – Llogarit bonusin me CASE WHEN

Krijonën një kolonë të re "bonusi" bazuar në shumën e shitjeve: nën 100 = 5%, 100–500 = 10%, mbi 500 = 15%.

SELECT emri, shuma, CASE WHEN shuma < 100 THEN shuma * 0.05 WHEN shuma BETWEEN 100 AND 500 THEN shuma * 0.10 ELSE shuma * 0.15 END AS bonusi, shuma + CASE WHEN shuma < 100 THEN shuma * 0.05 WHEN shuma BETWEEN 100 AND 500 THEN shuma * 0.10 ELSE shuma * 0.15 END AS shuma_finale FROM shitjet;

List Comprehension – Filtro numrat çiftë

Nga një listë e numrave 1–50, filtro vetëm ato çiftë dhe ngrita secilin në katror.

# Lista origjinale numrat = list(range(1, 51)) # List comprehension: çiftët në katror katrore_cifte = [ n ** 2 for n in numrat if n % 2 == 0 ] print("Numrat çiftë (katror):") print(katrore_cifte) print(f"Totali: {sum(katrore_cifte)}") print(f"Sa numra: {len(katrore_cifte)}")

Algoritëm – Renditja Bubble Sort

Implemento algoritmin Bubble Sort nga zeroja dhe testo me një listë të rastësishme.

import random def bubble_sort(lista): n = len(lista) lista = lista[:] # kopje for i in range(n): ndërruar = False for j in range(0, n-i-1): if lista[j] > lista[j+1]: lista[j], lista[j+1] = ( lista[j+1], lista[j] ) ndërruar = True if not ndërruar: break # optimizim i hershëm return lista # Test lista_random = [random.randint(1, 100) for _ in range(10)] print(f"Para: {lista_random}") print(f"Pas: {bubble_sort(lista_random)}")
