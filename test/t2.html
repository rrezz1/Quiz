SUM / TOTAL – Gjej totalin e shitjeve

Nga tabela shitjet, llogarit shumën totale të të gjitha shitjeve me SUM.

-- Totali i të gjitha shitjeve
SELECT
    SUM(shuma)   AS total_shitjet,
    COUNT(*)     AS nr_transaksioneve,
    AVG(shuma)   AS mesatarja
FROM shitjet;

-- Totali sipas muajit:
SELECT muaji, SUM(shuma) AS total
FROM shitjet
GROUP BY muaji
ORDER BY muaji;

GROUP BY – Shitjet sipas produktit

Gropo shitjet sipas emrit të produktit dhe gjej totalin për secilin produkt.

SELECT
    produkti,
    SUM(shuma)  AS total,
    COUNT(*)    AS nr_shitjeve
FROM shitjet
GROUP BY produkti
HAVING SUM(shuma) > 1000
ORDER BY total DESC;

Missing Numbers – Gjej numrat që mungojnë

Nga tabela porosite, gjej të gjithë numrat ID që mungojnë në sekuencë.

-- Metoda me LEFT JOIN (punon kudo):
SELECT t1.id + 1 AS id_mungues
FROM porosite t1
LEFT JOIN porosite t2
    ON t1.id + 1 = t2.id
WHERE t2.id IS NULL
ORDER BY id_mungues;

-- PostgreSQL me generate_series:
SELECT gs AS id_mungues
FROM generate_series(
    (SELECT MIN(id) FROM porosite),
    (SELECT MAX(id) FROM porosite)
) gs
WHERE gs NOT IN (SELECT id FROM porosite);

JOIN – Bashko dy tabela

Bashko porosite me klientet dhe shfaq emrin e klientit me shumën e porosisë.

SELECT
    k.emri,
    k.mbiemri,
    p.data_porosise,
    p.shuma_totale
FROM porosite p
INNER JOIN klientet k
    ON p.klient_id = k.id
WHERE p.shuma_totale > 500
ORDER BY p.data_porosise DESC;

Formula – Llogarit bonusin me CASE WHEN

Krijo kolonën "bonusi": nën 100 = 5%, 100–500 = 10%, mbi 500 = 15% të shumës.

SELECT
    emri,
    shuma,
    CASE
        WHEN shuma < 100               THEN shuma * 0.05
        WHEN shuma BETWEEN 100 AND 500 THEN shuma * 0.10
        ELSE                               shuma * 0.15
    END AS bonusi
FROM shitjet
ORDER BY bonusi DESC;

Formula – Llogarit bonusin me funksion

Krijo një funksion që llogarit bonusin e punonjësit bazuar në pagë dhe performancë.

def llogarit_bonusin(paga, performanca):
    perqindjet = {
        'e keqe':     0.00,
        'mesatare':   0.05,
        'shkelqyese': 0.15,
    }
    p = perqindjet.get(performanca.lower(), 0)
    return round(paga * p, 2)

# Test me listë punonjësish
punonjesit = [
    ("Alban", 80000, "shkelqyese"),
    ("Drita", 60000, "mesatare"),
    ("Besa",  70000, "e keqe"),
]
for emri, paga, perf in punonjesit:
    b = llogarit_bonusin(paga, perf)
    print(f"{emri}: bonus = {b} €")

Pandas – Analizë e shitjeve me DataFrame

Ngarko një CSV me shitje dhe gjej produktin me shitjet më të larta me Pandas dhe grupim.

import pandas as pd

# Ngarko të dhënat
df = pd.read_csv("shitjet.csv")

# Grupim sipas produktit
rezultati = (
    df.groupby("produkti")["shuma"]
      .agg(["sum", "count", "mean"])
      .rename(columns={
          "sum":   "totali",
          "count": "nr_shitjeve",
          "mean":  "mesatarja",
      })
      .sort_values("totali", ascending=False)
)

print(rezultati.head(10))
top = rezultati.index[0]
print(f"\nProdukti #1: {top}")

SQLite + Python – Krijo dhe query-o bazën

Krijo një bazë SQLite me Python, shto të dhëna dhe ekzekuto query me JOIN.

import sqlite3

conn = sqlite3.connect(":memory:")
cur  = conn.cursor()

cur.execute("""CREATE TABLE klientet (
    id    INTEGER PRIMARY KEY,
    emri  TEXT NOT NULL)""")

cur.execute("""CREATE TABLE porosite (
    id        INTEGER PRIMARY KEY,
    klient_id INTEGER,
    shuma     REAL,
    FOREIGN KEY (klient_id) REFERENCES klientet(id))""")

cur.executemany("INSERT INTO klientet VALUES (?,?)",
    [(1,"Alban"), (2,"Drita"), (3,"Besa")])

cur.executemany("INSERT INTO porosite VALUES (?,?,?)",
    [(1,1,250),(2,1,180),(3,2,99),(4,3,420)])

cur.execute("""
    SELECT k.emri, SUM(p.shuma) AS totali
    FROM klientet k
    JOIN porosite p ON k.id = p.klient_id
    GROUP BY k.emri
    ORDER BY totali DESC""")

for row in cur.fetchall():
    print(row)

conn.close()

List Comprehension – Filtro numrat çiftë

Nga numrat 1–50, filtro vetëm ata çiftë dhe ngrita secilin në katror.

# Lista origjinale
numrat = list(range(1, 51))

# List comprehension: çiftët në katror
katrore_cifte = [
    n ** 2
    for n in numrat
    if n % 2 == 0
]

print(f"Rezultati: {katrore_cifte}")
print(f"Totali:    {sum(katrore_cifte)}")
print(f"Sa numra:  {len(katrore_cifte)}")

Algoritëm – Bubble Sort nga zeroja

Implemento Bubble Sort dhe testo me një listë të rastësishme prej 10 numrash.

import random

def bubble_sort(lista):
    n     = len(lista)
    lista = lista[:]          # punojmë me kopje
    for i in range(n):
        nderruar = False
        for j in range(0, n - i - 1):
            if lista[j] > lista[j + 1]:
                lista[j], lista[j+1] = lista[j+1], lista[j]
                nderruar = True
        if not nderruar:
            break             # optimizim i hershëm
    return lista

# Test
lista_r = [random.randint(1, 100) for _ in range(10)]
print(f"Para: {lista_r}")
print(f"Pas:  {bubble_sort(lista_r)}")
